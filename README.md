# 🧠 Deep Learning Portfolio

Welcome to my **Deep Learning Portfolio**.

This repository documents my journey of learning and building Deep Learning projects from the ground up using **PyTorch**. Every project is built from scratch to strengthen my understanding of the underlying concepts rather than relying on high-level libraries or pre-trained solutions.

The goal of this repository is not only to build working models, but also to understand **how** and **why** they work.

# 🎯 Objectives

* Learn Deep Learning systematically.
* Build real-world Computer Vision projects.

# 🎓 Learning Philosophy

Rather than simply using pre-trained models, I focus on understanding every stage of the Deep Learning pipeline:

* Data preprocessing
* Model architecture
* Forward propagation
* Loss computation
* Backpropagation
* Optimization
* Validation
* Inference
* Deployment

Each project builds upon the previous one, creating a structured learning path from beginner concepts to production-ready AI systems.
